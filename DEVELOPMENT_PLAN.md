### 目标

- [ ] 配置 Electron 镜像源

- [ ] 验证 `npm run build` 可打包

- [ ] 安装 Claude Code CLI
- [ ] 配置 Agent 人格文件
- [ ] 测试 Agent 通信链路

#### 1.3 数据库完善 (P1)
- [ ] 验证 better-sqlite3 编译
- [ ] 实现数据迁移脚本
- [ ] 添加数据库索引优化
- [ ] 实现数据备份/恢复

#### 1.4 图像编辑器增强 (P1)
- [ ] 实现图层管理
- [ ] 添加滤镜效果
- [ ] 实现导出功能 (PNG/JPG)
- [ ] 集成 AI 图像生成

### 验收标准
- `npm run dev` 启动无错误
- Agent 可接收并响应指令
- 数据库读写正常
- 图像编辑器可导出图片

### 风险与应对
| 风险 | 概率 | 应对 |
|------|------|------|
| Electron 下载失败 | 中 | 配置国内镜像源 |
| better-sqlite3 编译失败 | 中 | 使用预编译二进制 |
| Claude CLI 不可用 | 低 | 提供 mock 模式 |
